頻出作業
###############################

不要な文字列データを削る
**************************

* プラン：正規表現でパターンマッチ → 置換する

tweet から抜き出したテキストから、URLを削りたい
-------------------------------------------------

* 対象文字列を確認する

::

  # URL だけを抽出する
  df['text'].str.findall('(https?://[a-zA-Z0-9;/?:@&=\+$,\-_\.!~*`\(\)%#]+)')

  # @メンションだけを抽出する
  df['text'].str.findall('(@[a-zA-Z0-9]+)')


* 文字列を削除する

::

  # URL を削除する
  df['text']=df['text'].str.replace("https?://[a-zA-Z0-9;/?:@&=\+$,\-_\.!~*`\(\)%#]+","")

  # @メンションを削除する
  df['text']=df['text'].str.replace("@[a-zA-Z0-9]+","")
